## Use Auto Upgrade For Bitcanna-Devnet

## Download script for auto upgrade
```
cd $HOME
wget https://raw.githubusercontent.com/konsortech/Node/main/Testnet/Bitcanna/upgrade/v1.6.1.sh
chmod +x v1.6.1.sh
```

## Use Screen
```
screen -S bitcanna-upgrade
./v1.6.1.sh
```
